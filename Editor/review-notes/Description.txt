这是一款通用的，了解不同文件格式的文件结构的工具。
对于格式已知的二进制文件，选择或者创建合适的grammar之后，分析引擎会根据grammar所提供的信息，自动解析二进制文件的内容，并将结果以树形结构显示出来。选择文件结构中树形节点，会在文件中高亮显示对应内容，同理，点击文件中的任意地方，会自动定位并展开到对应的树形结构。

This is a universal tool for understanding the file structure of different file formats.
For a binary file with a known format, after selecting or creating a suitable grammar, the analysis engine will automatically parse the content of the binary file according to the information provided by the grammar, and display the result in a tree structure. Selecting the tree node in the file structure will highlight the corresponding content in the file. Similarly, clicking anywhere in the file will automatically locate and expand to the corresponding tree structure.
